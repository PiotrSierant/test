### Head and meta tags

```javascript
Tytuł strony: <title>Tytuł strony</title>
Opis strony: <meta name="description" content="Opis strony">
Słowa kluczowe: <meta name="keywords" content="słowa, kluczowe">
Autor: <meta name="author" content="Imię i nazwisko autora">
Kodowanie znaków: <meta charset="UTF-8">
Skalowanie na urządzeniach mobilnych: <meta name="viewport" content="width=device-width, initial-scale=1.0">
Indeksowanie przez wyszukiwarki: <meta name="robots" content="index, follow">
Tytuł okna przeglądarki: <meta name="application-name" content="Tytuł aplikacji">
Ikona strony: <link rel="icon" type="image/png" href="sciezka/do/ikony">
Zastosowanie dla urządzeń Apple: <meta name="apple-mobile-web-app-capable" content="yes">
Odpowiednie dla wersji językowej: <meta http-equiv="content-language" content="pl-PL">
Adres URL kanoniczny: <link rel="canonical" href="https://www.przykładowastrona.pl">
Przypisanie do formatu: <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
Format daty: <meta name="date" content="YYYY-MM-DD">
Autorskie prawa: <meta name="copyright" content="Nazwa firmy">
Refreshing strony: <meta http-equiv="refresh" content="czas; URL">
Pamięć podręczna: <meta http-equiv="cache-control" content="public">
Google Search Console: <meta name="google-site-verification" content="kod weryfikacyjny">
Ochrona przed spamem: <meta name="spam-protection" content="UWAGA">
Dostępność: <meta name="rating" content="general">
Generator treści: <meta name="generator" content="Nazwa narzędzia">
Referencyjny adres URL: <meta name="referrer" content="unsafe-url">
```
